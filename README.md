# Recursive-Problems (C#)

A collection of implementations for classic recursive problems using **Backtracking** techniques. Built with **.NET 8.0**.

## 🧠 Featured Algorithms

### **1. Traveling Salesperson Problem (TSP)**
* Finding the shortest possible route that visits 10 cities exactly once and returns to the origin

### **2. N-Queens Problem**
* Employes a recursive backtracking algorithm to place $N$ queens on an $N \times N$ chessboard such that no two
  queens threaten each other

### **3. Knight's Tour**
* A mathematical challenge where the object is to find a sequence of moves for a knight on a chessboard such that it
  visits every square exactly once
  
### **4. 0/1 Knapsack Problem**
* A fundamental combinatorial problem. Given a set of items with specific weights and values, the goal is to determine
  the most valuable combination to fit in a knapsack of limited capacity
